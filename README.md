basic.py: 맵 생성
main.py: 실행 환경 설정(래퍼 클래스 작성함)
src/planner.py: 경로 생성 + 제어 함수
src/perception.py: 영역 감지 + 장애물 처리된 배열 반환 함수
Examples: 실행 파일들

계획
눈 감지 후 맵 생성 함수 구현 -> 경로 생성 후 이동 시뮬레이션 /  -> 제어 관련 함수 작성(시간이 된다면)
함수 구현되면 맵 로드 함수만 새로 만들어서 파일 작성 하면 됨

사용법
첫 번째 좌클릭: 로봇 초기 위치
두 번째 좌클릭: 목표
세 번째: 맵 생성(장애물 그리기 가능)
